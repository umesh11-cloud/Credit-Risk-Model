📊 Credit Risk Model

A Machine Learning-based Credit Risk Prediction System that evaluates the probability of a borrower defaulting on a loan. This project helps financial institutions make smarter, data-driven lending decisions.

🚀 Overview

The Credit Risk Model analyzes customer financial and behavioral data such as income, loan amount, credit history, and repayment patterns to classify borrowers into low-risk or high-risk categories.
